# Dynamodb to ElasticSearch

> NB: Before using Dynamodb, think about it.

## Requirement

* [x] NodeJS.
* [x] ElasticSearch.
* [x] A brain.
* [x] oh.. and Yarn

## Start the party

```
yarn i
mv .env.default .env
```

Change the configuration in .env

```
yarn run start
```

## Test 

```
yarn run test

```
